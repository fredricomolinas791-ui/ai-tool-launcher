# AI Tools Launcher - 规格说明

## 1. 项目概述

- **项目名称**：AI Tools Launcher
- **类型**：Web 应用 - 工具导航页面
- **核心功能**：展示和管理多个AI工具的入口页面
- **目标用户**：需要快速访问各种AI工具的用户

## 2. UI/UX 规格

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  LOGO          [搜索栏]              用户信息     │  <- 顶部 Header (56px)
├──────────┬────────────────────────────────────────┤
│          │                                        │
│  左侧导航 │           主内容区                    │
│  (240px) │         (剩余宽度)                    │
│          │                                        │
│          │   ┌─────┐ ┌─────┐ ┌─────┐             │
│          │   │卡片1│ │卡片2│ │卡片3│             │
│          │   └─────┘ └─────┘ └─────┘             │
│          │   ┌─────┐ ┌─────┐ ┌─────┐             │
│          │   │卡片4│ │卡片5│ │卡片6│             │
│          │   └─────┘ └─────┘ └─────┘             │
└──────────┴────────────────────────────────────────┘
```

- **左侧导航**：固定宽度 240px
- **右侧主内容**：flex: 1，自适应
- **响应式**：移动端隐藏侧边栏，汉堡菜单

| 用途 | 色值 |
|------|------|
| 背景（主） | `#0a0a0f` |

- **主字体**：Inter, system-ui, sans-serif
- **数字/代码**：JetBrains Mono, monospace

#### 间距系统
- 基础单位：4px
- 卡片间距：24px (gap-6)
- 内边距：16px / 24px

### 组件规格

#### 左侧导航项
- 高度：44px
- 圆角：12px
- 图标：20px
- 状态：
  - 默认：文字 `#9ca3af`，无背景
  - 悬浮：背景 `#1a1a24`，文字 `#ffffff`
  - 选中：背景 `#6366f1/20`，左边框 3px 主色，文字 `#6366f1`

#### 工具卡片
- 尺寸：自适应，最小 280px 宽
- 高度：160px
- 圆角：16px
- 结构：
  - 图标：40px，背景 `#1a1a24` 圆形
  - 标题：18px，font-weight 600
  - 描述：14px，color 次级灰
  - 悬浮效果：Y轴位移 -4px，阴影扩散，边框发光

#### 搜索栏
- 高度：44px
- 圆角：12px
- 背景：`#13131a`
- 边框：1px `#2a2a3a`
- 聚焦：边框变为主色
- 占位符："搜索工具..."

#### 用户头像
- 尺寸：36px 圆形
- 背景：主色调渐变

## 3. 功能规格

### 交互行为

1. **导航切换**
   - 点击左侧导航，主内容区卡片刷新
   - 支持键盘导航（上下键+回车）

2. **搜索过滤**
   - 实时过滤卡片（debounce 150ms）
   - 高亮匹配文字

3. **卡片点击**
   - 显示提示信息（功能开发中）
   - 涟漪动画

4. **主题切换**（可选）
   - 深色/浅色切换按钮
